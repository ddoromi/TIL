## 큐

#### 1. 특성

* 스택과 마찬가지로 삽입과 삭제의 위치가 제한적인 자료구조
  - 큐의 뒤에서는 삽입만 하고, 큐의 앞에서는 삭제만 이루어지는 구조
* 선입선출구조
  - 큐에 삽입한 순서대로 원소가 저장되어, 가장 먼저 삽입된 원소는 가장 먼저 삭제 된다.

```python
QSIZE = 100
Q = [0] & QSIZE
front = rear = 1

def push(item):
    global rear
    rear += 1
    Q[rear] = item
def pop():
    global front
    front += 1
    return Q[front]
def empty():
    return front == rear
for i in range(5):
    push(i)
while not empty():
    print(pop())
```
